## Tips mengambil screenshot

- Pakai **Chrome/Edge**, jendela ukuran desktop (≥ 1280 px) agar tata letak rapi.
- Untuk satu komponen (mis. modal/kartu), pakai *device toolbar* atau crop setelahnya.
- Hindari menampilkan data sensitif/asli; gunakan data demo dari seeder.
